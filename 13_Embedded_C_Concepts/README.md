# 13 - Embedded C Concepts

This is where C meets hardware! These concepts are
unique to embedded programming and heavily tested in interviews.

## Files

| File | Topic |
|------|-------|
| 01_volatile_keyword.c | volatile — #1 embedded interview topic |
| 02_const_keyword.c | const + Flash storage + pointer const |
| 03_register_keyword.c | register, storage classes |
| 04_interrupt_basics.c | ISR patterns, volatile flags, critical sections |
| 05_timer_concept.c | Timer registers, period calculation |
| 06_pwm_concept.c | Duty cycle, LED brightness, servo |
| 07_memory_mapped_io.c | Accessing hardware via pointers |

## Top Interview Questions from This Folder
1. What is `volatile` and when do you use it?
2. What is the difference between `const` and `#define`?
3. What is an ISR? What should/shouldn't you do inside it?
4. What is Memory-Mapped I/O?
5. How do you set/clear a bit in a register?
6. What is PWM? How do you calculate duty cycle?
