# 02 - Operators in C

This folder covers all types of operators in C.
Pay special attention to bitwise operators — they're used ALL the time in embedded!

## Files

| File | Topic |
|------|-------|
| 01_arithmetic_operators.c | +, -, *, /, % |
| 02_relational_operators.c | ==, !=, >, <, >=, <= |
| 03_logical_operators.c | &&, ||, ! |
| 04_assignment_operators.c | =, +=, -=, &=, |=, ^= |
| 05_increment_decrement.c | ++, -- (pre vs post) |
| 06_ternary_operator.c | condition ? a : b |

## Interview Tip
- Difference between `i++` and `++i` is a very common interview question!
- Bitwise assignment operators (`|=`, `&=`, `^=`) are heavily used in GPIO register manipulation
