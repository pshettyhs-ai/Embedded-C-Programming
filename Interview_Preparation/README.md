## Tips from My Preparation

1. **Bitwise operations** — practice until you can do it without thinking
2. **volatile and const** — always explained in every interview
3. **Pointers** — pointer to function, void pointer, const pointer
4. **UART/SPI/I2C** — know the protocol basics, transaction formats
5. **Interrupts** — what should and shouldn't be in an ISR
